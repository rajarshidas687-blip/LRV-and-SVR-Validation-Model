# GSR Device Validation Study

This repository contains Python code for validating GSR IC and GSR Module using:

- Direct statistical comparison
- Linear Regression equivalence
- Leave-One-Subject-Out SVR validation
- Bland-Altman analysis

## Requirements
Python 3.10
pandas
numpy
matplotlib
scikit-learn
Save the combined dataset from the report as name "GSR_data.xlsx"
Update your file path in code (Generally - "C:\GSR_data.xlsx")
